alumbra.web.graphiql-workspace

Ring handler for exposing the GraphiQL Workspace

handler

(handler options)

Create a Ring Handler exposing the GraphiQL Workspace web UI. Options are:

  • :title: content for HTML title tag.
  • :versions: versions of CSS/JS to fetch from CDN (jsDelivr) with keys :bootstrap, :graphiql, :graphiql-workspace and :react.
  • :config: map passed to GraphiQL Workspace’s AppConfig.

For example, to preconfigure a URL and query, you’d create a handler like:

(handler
  {:config
   {:defaultUrl   "https://some.graphql.endpoint/graphql"
    :defaultQuery "{ awesomeness { totalCount } }"}})
